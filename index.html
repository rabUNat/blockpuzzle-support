サポート

困ったときに確認してほしい内容をまとめています。

遊び方

都道府県ごとのチャレンジを選び、手数制限の中で条件達成を目指す思考型ブロックパズルです。

コイン要素

チャレンジ達成や広告視聴、アプリ内課金でコインを獲得し、テーマや背景の変更に使えます。

サポート機能

ミス取り消しや手数追加など、一部の補助機能はコインを消費して利用できます。

よくある質問

App Store 審査時にも説明しやすい内容に絞っています。

コインが反映されない

通信状態や決済処理の反映に少し時間がかかる場合があります。時間を置いても反映されない場合は、問い合わせ欄の案内からご連絡ください。

広告が読み込まれない

通信環境や広告配信状況により、広告が表示されないことがあります。時間を置いて再度お試しください。

ゲームの進行状況について

プレイ中に地図へ戻ると、そのチャレンジの進行状態はリセットされます。各チャレンジは再挑戦可能です。

問い合わせ

連絡導線と、同ページ内で見られるプライバシーポリシーを用意しています。

連絡方法

問い合わせや不具合報告は、GitHub リポジトリの Issues から受け付けます。

  • 不具合の内容
  • 発生した画面や操作手順
  • 利用している iPhone / iOS バージョン
  • 課金や広告に関する問題の場合は、その状況

プライバシーポリシー

ブロックパズル全国制覇では、サービス提供と品質向上のために、必要な範囲で情報を取り扱います。

1. 取得する可能性がある情報

  • アプリの利用状況やクラッシュに関する情報
  • 広告表示に必要な識別情報や配信関連情報
  • アプリ内課金の購入状況に関する情報

2. 利用目的

  • アプリの提供、動作改善、不具合対応
  • 広告表示と不正利用防止
  • アプリ内課金の処理と購入内容の反映

3. 外部サービス

本アプリでは、広告配信およびアプリ内課金のために第三者サービスを利用する場合があります。

  • Google AdMob
  • Apple App Store / StoreKit

4. 個人情報の第三者提供

法令に基づく場合を除き、開発者が個人情報を第三者へ任意に提供することはありません。

5. お問い合わせ

本ポリシーに関する問い合わせは、上記の GitHub Issues から受け付けます。

6. 改定

必要に応じて本ポリシーを改定することがあります。改定後はこのページに掲載した時点で効力を生じます。
